# Installed libraries

## UI & styling

- **Tailwind CSS** – utility-first CSS framework
- **Radix UI** – accessible UI primitives (dropdowns, dialogs, sliders, etc.)
- **lucide-react** – icon library

## Internationalization

- **i18next** – core library for i18n
- **react-i18next** – React bindings for i18n

## Forms (to be added)

- **react-hook-form** – recommended for login and registration forms
- **yup / zod** – optional, for form validation schemas
